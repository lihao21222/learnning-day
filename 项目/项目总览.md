## 📦 项目说明

### 项目一：Local RAG（本地知识库问答系统）

**面向场景**：个人/小团队的本地文档智能问答

**三个版本**：
| 版本 | 文件 | 特点 | 复杂度 |
|------|------|------|--------|
| 简单版 | simple_rag.py | 演示 RAG 核心概念，代码最少 | ⭐ |
| 进阶版 | advanced_rag.py | 模块化、配置管理、日志系统 | ⭐⭐ |
| 专业版 | rag_pro.py + document_loader_pro.py | 多格式支持、策略模式设计 | ⭐⭐⭐ |

**快速启动**：
```bash
cd local-rag
python demo_advanced.py    # 命令行演示
streamlit run app_simple.py  # Web 界面
```

---

### 项目二：SupplyGuard（供应链风控 Agent 平台）

**面向场景**：企业级供应链风险监控

**核心架构**：
- Multi-Agent 系统（4个 Agent 协同）
- FastAPI 服务接口
- Streamlit 可视化界面
- 可审计的决策流程
- 生产级架构设计

**快速启动**：
```bash
cd supplyguard-standalone
python demo.py            # 命令行演示
python main.py            # 启动 API 服务
streamlit run app.py      # 启动 Web 界面
```
