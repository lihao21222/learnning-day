# 本地知识库问答系统 - 版本说明

## 📁 三个版本对比

| 版本 | 主文件 | 复杂度 | 特点 | 适合场景 |
|------|--------|--------|------|----------|
| 基础版 | simple_rag.py | ⭐ | 极简实现，核心概念清晰 | 学习和演示 |
| 进阶版 | advanced_rag.py | ⭐⭐ | 模块化、配置、日志 | 实际项目使用 |
| 专业版 | rag_pro.py | ⭐⭐⭐ | 多格式支持、策略模式 | 生产级项目 |

---

## 🚀 快速开始

### 基础版
```bash
python simple_rag.py
```

### 进阶版
```bash
python demo_advanced.py              # 演示
streamlit run app_simple.py          # Web 界面
```

### 专业版
```bash
python demo_pro_final.py             # 演示（支持 PDF/Word）
```

## 🎯 面试建议

### 讲项目时的推荐顺序
1. **先讲架构** - 可以从基础版讲起，再讲进阶版的改进
2. **展示代码** - 重点讲模块化和设计模式
3. **演示功能** - 启动 Web 界面给面试官看
4. **扩展规划** - 讲如何集成真实的 LLM、向量数据库

### 重点讲的内容
- 如何做文档切分
- 如何实现语义检索
- 如何优化搜索效果
- 如何工程化（配置、日志、测试）
