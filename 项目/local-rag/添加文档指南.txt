如何向知识库添加文档
================================

方法一：直接放入文件夹（最简单）
----------------------------------
1. 把您的文档（.txt格式）复制到 knowledge_base 文件夹中
2. 重新运行程序即可
3. 例如：您可以创建一个 my_notes.txt，把内容写进去，然后放进去


方法二：Web 界面上传（推荐）
----------------------------------
1. 安装 streamlit: pip install streamlit
2. 运行: streamlit run app_simple.py
3. 在浏览器中打开页面
4. 在左侧边栏上传您的文档


方法三：手动输入（测试用）
----------------------------------
如果只是想测试，可以直接在 Web 界面的"手动输入"框里输入内容


支持的格式
----------------------------------
- .txt - 纯文本（推荐，无需额外安装）
- .pdf - 需要安装: pip install pypdf
- .docx - 需要安装: pip install python-docx


文档内容建议
----------------------------------
- 格式清晰，有标题和段落
- 一个主题一个文件，或者都放在一个文件里也可以
- 中文英文都可以
