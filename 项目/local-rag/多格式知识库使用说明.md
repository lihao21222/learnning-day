### 1. 运行专业版演示

```

```

### 3. 查看当前知识库

```bash

```

## 📄 添加文档的方法

### 方法一：直接放入文件夹（推荐）

将您的文档文件复制到 `knowledge_base/` 文件夹，然后重新运行程序即可。

支持的文件：
- `您的文档.txt`
- `您的文档.docx`
- `您的文档.pdf`

### 方法二：使用 Web 界面上传

```bash
pip install streamlit  # 如果还没安装
streamlit run app_simple.py
```
