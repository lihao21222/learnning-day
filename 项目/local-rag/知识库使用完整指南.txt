一、如何添加文档
================================================================================

方法一：直接放入文件夹（最简单）
----------------------------------
1. 找到 knowledge_base 文件夹
2. 把您的 .txt 文件复制进去
3. 重新运行程序即可
4. 支持多个文件，会全部加载


方法二：交互式添加（测试用）
----------------------------------
运行: python advanced_rag.py
然后您可以输入问题，但添加新文件需要重启程序


方法三：Web 界面（推荐，最直观）
----------------------------------
1. 安装依赖: pip install streamlit
2. 运行: streamlit run app_simple.py
3. 在网页上可以直接上传文件或手动输入


二、支持的文档格式
================================================================================

✅ .txt - 纯文本（推荐，无需额外安装）
⚠️  .pdf - 需要安装: pip install pypdf
⚠️  .docx - 需要安装: pip install python-docx


三、如何开始使用
================================================================================

步骤 1：准备文档
--------------
把您的文本文件放到 knowledge_base 文件夹里

步骤 2：运行演示
--------------
python demo_advanced.py

步骤 3：交互式问答
--------------
python advanced_rag.py

步骤 4：Web 界面（可选）
--------------
pip install streamlit
streamlit run app_simple.py

六、自定义修改
================================================================================

修改配置（在 config.py 中）：
- CHUNK_SIZE = 500  # 每个文档块的大小
- TOP_K = 3         # 检索返回的相关文档数


七、项目文件说明
================================================================================

基础功能：
- simple_rag.py - 简化版，代码最少，适合学习
- demo.py - 简化版演示

进阶功能：
- advanced_rag.py - 完整版，架构清晰，面试推荐
- demo_advanced.py - 完整版演示
- document_loader_pro.py - 支持多种格式的文档加载器
- config.py - 配置文件
- logger.py - 日志系统

Web 界面：
- app_simple.py - Streamlit 网页界面
- start_web.bat - 一键启动网页（Windows）
- requirements-full.txt - 完整版依赖

面试相关：
- 面试项目介绍_豪华版.md - 项目介绍，面试时讲这个
- 面试展示指南.md - 面试技巧和展示要点

九、常见问题
================================================================================

Q: 如何添加更多文档？
A: 把 .txt 文件放到 knowledge_base 文件夹即可

Q: 如何清空知识库？
A: 删除 knowledge_base 文件夹里的文件，或者用 Web 界面的清空按钮

Q: 如何切换到更好的检索模型？
A: 目前是关键词匹配，后期可以添加 sentence-transformers 和 Chroma

Q: 如何在面试中展示？
A: 推荐用 Web 界面，视觉效果好，操作简单


十、下一步（可选的优化）
